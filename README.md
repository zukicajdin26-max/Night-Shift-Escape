# 😱 Night Shift Escape

Night Shift Escape is a simple horror survival game where you must escape a building after your coworkers suddenly turn hostile.

What started as a normal night shift quickly becomes a nightmare. Strange sounds fill the halls, and the workers are no longer human. They are hunting you.

---

## 🎮 Gameplay

* Move using **W A S D**
* Avoid enemies chasing you
* Find the exit to escape
* If they catch you… it's game over
